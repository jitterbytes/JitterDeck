# JitterDeck

A portable embedded hardware validation tool for testing sensors,
communication interfaces, and development boards without writing
dedicated firmware.

## Features

- I²C Scanner
- Sensor Profiles
- UART Monitor
- ADC Monitor
- WiFi Scanner
- BLE Scanner

## Hardware

- ESP32-C3 SuperMini  
- 1.3" SH1106 OLED  
- 3 Navigation Buttons
